# EarlyDrop: Multimodal Student Dropout Risk Prediction System

## Overview

EarlyDrop is a multimodal machine learning system developed for the **Retina AI: Predict Student Dropout Risk** challenge.

The project predicts whether a student belongs to a **Low Risk**, **Medium Risk**, or **High Risk** dropout category by combining information from multiple data sources including academic records, attendance trends, and counsellor notes.

By integrating structured, time-series, and text data, EarlyDrop enables early identification of at-risk students and supports proactive intervention strategies within educational institutions.

### Key Highlights

* Multimodal Machine Learning Pipeline
* Academic + Attendance + NLP Feature Fusion
* ~198 Engineered Features
* CatBoost-Based Classification Model
* 5-Fold Stratified Cross Validation
* Macro F1 Score: **0.7141**
